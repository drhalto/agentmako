# Roadmap Version 1

**Status:** ✅ COMPLETE - All phases shipped (0-5)

This directory contains the historical Roadmap 1 documentation. Roadmap 1 delivered the clean local-first rebuild and the first real shipped surface of `mako-ai`.

## What Roadmap 1 Delivered

- Local-first repo indexing
- Shared typed tool contracts
- MCP, HTTP, CLI, and thin web surfaces
- 19 structured tools plus `ask` router
- Read-only database schema tools
- Public-facing install and tool docs
