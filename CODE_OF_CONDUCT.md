# Code of Conduct

The agentmako project adopts the [Contributor Covenant, version 2.1][cc]
as its code of conduct. The full text is available at the link below.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## In short

We want agentmako to be a welcoming project for contributors of any
background and experience level. Be respectful in issues, pull requests,
discussions, and any other project space. Disagree on technical merits,
not on people.

If you observe behavior that violates the Contributor Covenant, please
report it through one of the channels in the next section. Reports are
handled confidentially.

## Reporting

- **Preferred:** open a private report through the repository's
  [security advisory form](https://github.com/drhalto/agentmako/security/advisories/new).
  Conduct reports filed there are routed to the maintainers privately
  even though the form is named for security issues.
- **Alternative:** contact the repository owner directly via their
  GitHub profile.

Please do not file public issues for conduct reports.

## Scope

This code of conduct applies to all project spaces — the GitHub
repository (issues, pull requests, discussions, code review), any
official chat or community space, and any direct interaction where you
are representing the agentmako project.

## Enforcement

Maintainers will follow the Contributor Covenant's
[Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines)
when responding to reports. Possible responses range from a private
correction to a temporary or permanent ban from project spaces,
proportional to the severity and pattern of the behavior.
