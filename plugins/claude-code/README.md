# Mako AI for Claude Code

This is the new generated plugin layout for Claude Code. The published stable
path remains `mako-ai-claude-plugin/` until the marketplace listing moves.

Validate or run from the repo root:

```bash
claude plugin validate ./plugins/claude-code
claude --plugin-dir ./plugins/claude-code
```

The MCP server runs through `npx -y agentmako mcp`.
